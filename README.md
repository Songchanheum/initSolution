# initSolution

### 구성
 1. Server - Spring Boot, JPA
 2. Client - ReactJS, ChakraUI


### 실행 방법
1. Server 실행 - Spring Boot
 - maven update 후 Spring Boot start
 - 초기 port : 9999 

2. Client 실행
 - frontend directory 진입 
 
 ``` $ cd frontend ```
 - yarn install
 
 ``` $ yarn install ```
 - Client 실행
 
 ``` $ yarn start ```
